Review all collected data and surface patterns, connections, and gaps.

Execute the /q-analyze steps from `.q-system/commands.md`.

1. Read all target profiles in `investigation/targets/`
2. Read all findings in `investigation/findings/`
3. Read timeline in `investigation/timelines/`
4. Identify: connections, patterns, gaps, contradictions, new leads
5. For each finding, assess: confidence level, supporting evidence, alternative explanations
6. Update `investigation/findings/` with new assessments using the template from `investigation/findings/_TEMPLATE.md`
7. Update `canonical/collection-plan.md` with new requirements
8. Present analysis to user

Argument: $ARGUMENTS (optional: specific aspect to focus analysis on)
